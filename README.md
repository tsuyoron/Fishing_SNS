# 釣果シェア

## サイト概要
### サイトテーマ
釣果をシェアして、どこでどのような魚が釣れているのかを共有できるサイト

### テーマを選んだ理由
釣りに行く際に、どこでどのような魚が釣れているかが分かると、便利だと考えたため。
また、釣り人同士での情報共有をしたいとかんがえたため。

### ターゲットユーザ
- 釣りが好きな人
- 釣りに興味がある人
- どの場所でどんな魚が釣れるか気になっている人
- 今の時期何が釣れるか気になる人

### 主な利用シーン
- 釣った魚を共有したい場合
- どこでどのような魚が釣れているのか気になるとき
- 釣り仲間と、情報共有したい場合
